# Hexadia

Hexadia is a custom Open Tibia Server based on the Tibia 8.0 protocol.

## About

Hexadia focuses on:
- authentic Tibia 8.0 gameplay
- custom content
- stable infrastructure
- long-term development
- balanced progression

## Features

- Custom map areas
- Unique quests
- Improved PvP systems
- Modern server architecture
- Active development
